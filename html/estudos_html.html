O que é HTML?

HTML, nada mais é do que uma limguagem de marcação que dá a estrutura de todas as paginas web.

As siglas HTML, vem de HyperText Markup Language, traduzindo: "Linguagem de Marcação de Hipertexto."

História do HTML

O HTML (HyperText Markup Language), foi criado pelo físico britânico Tim Berners-Lee, com ajuda do estudante do CERN (European Council for Nuclear Reserch), Robert Cailliau, em meados de 1989, quando eles conseguiram efetuar a primeira comunicação entre cliente HTTP e um servidor.

Pode-se afirmar que o HTML, surgiu no início da década de 90, entre 90 e 95 sofreu diversas atualizações e modificações estruturais. O controle e supervisão da linguagem foi do CERN até o ano de 1995, após esse período a W3C (The World Wide Web Consortium), entidade que regula os padrões da Web, tomou as rédias da linguagem.

Declaração <!DOCTYPE html>.

Começaremos pelo início, no nosso caso a tag <!DOCTYPE html>.

Primeiro apresentarei um trecho da estrutura de uma pagina HTML5.

<!DOCTYPE html>
<html>
<head>
<title> Aqui vai o Titulo do Documento. </title>
</head>

<body>
</body>
</html>

Definição e Uso

Todo o documento html deve iniciar com uma declaração, em HTML 5 utilizamos a declaração <!DOCTYPE html> .

É utilizado essa marcação, para mostrar para o navegador que estamos trabalhando com HTML na sua versão 5,
dessa forma o navegador poderá carregar a pagina utilizado as funcionalidades da versão.

Ao iniciar um projeto em HTML, declaramos a sua versão usando !DOCTYPE html, como mostrado no exemplo,
lembrando que a declaração !DOCTYPE não é uma tag, sendo assim ela não possue fechamento, existem tags que não possuem
fechamento, veremos no decorre dos estudos.

Declaração no HTML 4.01

Antes da versão do HTML 5, todo documento HTML deveria ser validado por uma DTD (Document Type Definition), traduzindo: Definição de Tipo de Documento. Para quê o ducumento seja devidamente validado, era preciso iniciar o documento com uma DTD correta na primeira linha.

A DTD Strict da versão 4.01 inclue elementos e atributos aprovados ou que não aparecem em conjuntos de molduras (framsets):

<!DOCTYPE HTML PUBLIC
"-//W3C//DTD HTML 4.01//EN"
"http://www.w3.org/TR/html4/strict.dtd">

A DTD Transitional do HTML 4.01, inclue toda DTD Strict mais os elementos e os atributos desaprovados.

<!DOCTYPE HTML PUBLIC
"-//W3C//DTD HTML 4.01 Transitional//EN"
"http://www.w3.org/TR/html4/loose.dtd">

A DTD Frameset do HTML 4.01 inclue toda DTD Transitional mais as molduras (frames).

<!DOCTYPE HTML PUBLIC
"-//W3C//DTD HTML 4.01 Frameset//EN"
"http://www.w3.org/TR/html4/frameset.dtd">
